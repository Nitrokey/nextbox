

- build the image `make`

- boot device, connect SSO account
- set up network
- ssh into the device (using key registered at ubuntu-core)

- sudu su for root account access

- add /etc/writable/hostname
- add /etc/systemd/system/media-nextcloud.mount
- add /etc/systemd/system/nextbox-daemon.service

- systemctl daemon-reload

- systemctl start media-nextcloud.mount
- systemctl enable media-nextcloud.mount

- systemctl start nextbox-daemon.service
- systemctl enable nextbox-daemon.service

- snap connect nextcloud-nextbox:removable-media
